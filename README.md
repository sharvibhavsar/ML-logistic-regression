# ML-logistic-regression
Machine Learning classification model using Logistic Regression with data preprocessing, EDA and Classification Model.

Demonstrates data preprocessing, exploratory data analysis, feature engineering, and supervised machine learning using Logistic Regression to predict target outcomes from structured data.

**Workflow:**

-Data cleaning and preprocessing

-Exploratory Data Analysis (EDA)

-Feature engineering

-Logistic Regression model training

-Model evaluation

**Tech Stack**

Python, Pandas, NumPy, Scikit-learn, Matplotlib

**Project Structure**

project.ipynb — main notebook

loan_data_set.csv — dataset
